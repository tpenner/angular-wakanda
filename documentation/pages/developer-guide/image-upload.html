Image upload

Image upload with Angular-Wakanda is realy straightforward. When you have an attribute of type image on a dataClass, this attribute will have a $upload() method available.

Uploading an image

We will use a simple file input with an identifier, and Angular-Wakanda will do the rest.

The image is then saved on Wakanda Server, and you can access it with an URI.

Retrieving image

Image is accessible with a URI, you can get it on the image attribute of the entity. 

ds.Person.$find(8).$promise.then(function (e) {
  var person = e.result;

  person.photo.uri; //http://...
});
